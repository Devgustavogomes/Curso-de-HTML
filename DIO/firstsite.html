Tags de texto

(Voltar)

Para fazer um texto de um site existem várias tags que podem ser utilizadas, como a tag "p" que serve para criar parágrafos, temos também as tag h1,h2,h3,h4 que servem para digitar uma frase ou texto com determinado tamanho quanto menor o número maior a fonte da letra, temos outras tags como "i" ou "strong" que servem para estilizar seu texto uma para deixar em formato itálico e outra em negrito respectivamente.

Lista ordenada/não ordenada

(Voltar)

Existem dois tipos de lista, a lista ordenada e a não ordenada, na lista ordenada se usa a tag "ol" para criar estilizar e a tag "li" para colocar itens nessa lista, a não ordenada é basicamente a mesma coisa invez de da tag "ol" se usa a "ul", oque difere uma da outra é q a ordenada como o nome indica vai ordenando os itens como 1,2,3 ... e a não ordenada fica apenas pontos separando cada item.

Links

(Voltar)

para usarmos links em seu site utilizamos a tag "a" e dentro dessa tag usamos o "href" com esse link podemos entrar em outro sites ou até mesmo fazer "âncoras" dentro do seu próprio site, podemos personalizar esse link para ser aberto na mesma aba ou em outra aba.
